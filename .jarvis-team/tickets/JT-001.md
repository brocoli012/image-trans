# JT-001: 이미지 일괄 변환 웹앱 (HEIC→JPEG) MVP + 배포

- **생성**: 2026-05-30
- **복잡도**: 4점 (web / greenfield / 파일 처리 / HEIC 디코딩)
- **모드**: Teams / Stage B (Iron Gate 해제)
- **위임 경로**: Master → TPO(설계) → Developer(구현) → QA(검증) → 배포

## 목표
부모님도 쉽게 쓰는, 여러 이미지(HEIC 필수)를 JPEG로 일괄 변환·다운로드하는 정적 웹 서비스. 클라이언트(브라우저) 변환.

## 결정 사항 (사용자 승인)
- 변환 위치: **클라이언트(브라우저)** — 100건 일괄, 업로드 0, 개인정보 보호, 비용 0
- 타겟 포맷: **JPEG/JPG만 (MVP)** — 포맷 드롭다운 제거 (F2 불필요 기능 감사)
- 스택: **React + Vite (정적)** — 자비스팀 재량
- 배포: 웹 배포까지

## 수용 기준 (AC)
1. HEIC 파일이 JPEG로 정상 변환·미리보기·다운로드
2. 여러 장(HEIC+일반 혼합) 일괄 변환, 최대 100건 진행률 표시
3. 변환 실패 파일은 건너뛰고 사유 표시 (실패 격리)
4. "전체 ZIP 다운로드" 1버튼 + 개별 다운로드
5. 동선 ≤3클릭, 큰 버튼/큰 글씨, 한글 안내 (부모님 친화, F3)
6. `npm run build` 성공 + 실제 브라우저에서 HEIC→JPEG 동작 검증
